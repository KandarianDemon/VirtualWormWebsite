Drag Physics

CyberElegans/MuscleSystem/DragPhysics

Description

inerhits from UnityEngine.Monobehaviour

Provides Implementation for physical forces applied to a cylindrical object moving in a gel like environment, as decribed by Resistive Force Theory.

Drag forces to a cylindrical body are anisotropous, meaning the tangential drag is less than orthogonal drag. In this implementation it facilitates directed propulsion of the worm through oscillating body bends.
