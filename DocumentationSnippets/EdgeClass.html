Edge

CyberElegans/wormbrain/Graph/Edge

Description

Inerhits from UnityEngine.MonoBehaviour

This class is used to represent the edge between two Nodes visible in the scene. It uses the positions of the nodes as start and end points.

A LineRenderer, which is attached to this GameObject renders a line. The main taks of this class is to render the connection and to update the start and end positions, should the Nodes have been moved through physics or user interaction

Properties
Type Name Description
Rigidbody start Reference to the rigidbody of the start node
Rigidbody end Reference to the rigidbody of the end node
Vector3 startPos Position of the start node
Vector3 endPos Position of the end node
LineRenderer renderer Renders the edge
bool isStatic determines wether physics is active or not and therefore, if the lines must be updated
float lineWidth strength of the line being drawn
Methods
Type Name Description
void InitEdge(Rigidbody start, Rigidbody end, Dictionary`<`string,NodeProperties> props, float strength, Material lineMaterial) Initializes the edge.
void SetActive(bool active) Sets the edge active or inactive. If inactive, edge won't be rendered and also will not effect force directed graph
void UpdatePosition() Updates start and end positions of the line, should the corresponding nodes have been moved
