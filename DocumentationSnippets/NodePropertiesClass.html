NodeProperties

CyberElegans/wormbrain/Graph/NodeProperties

Description

Object to store and manage data associated with the nodes or the represented neuron

Properties
Type Name Description
string name Represents the name of the node/neuron/ data represented by the node
string type represents the type of the neuron represented (sensory, interneuron,motoneuron)
Vector3 position Anatomical position of the node/neuron
Vector3 associatedPosition Position emphasized withr espect to dorso-lateral position. Position is offset according to the neurons being associated with dorsal, ventral, left or right side of the worms body
List`<`string> location
Color nodeColor Stores the color of the node
List`<`string> Edges Stores all edges starting with this node
Methods
Type Name Description
NodeProperties(string name) Constructor for a Nodeproperties object. Creates new, empty NodeProperties
void SetPosition(Vector3 newPosition) changes position to new position
void SetLocation(string[] locationIdentifier) Adds location tags to the location-list. Meaning d,v,l,r in order to identify how the neuron is associated laterally
