Graph Container

Namespace/Path/To/Class

Description

The Graph container class is used to turn a Graph into an interactive, three dimensional representation of the data.

Each Vertex in the Graph is turned into Node Game Object and is represented by a Sphere

Properties
Type Name Description
Graph graph Holds the connectivity data as a Graph
Dictionary`<`string,NodeProperties> props Dictionary containing a NodeProperties object for each Node.
int RecursionLevel Stores the current RecursionLevel, meaning the number of skip-connections between connected neurons, that are highlighted
bool fromMatrix Determines, wether the graph is constructed from a connectivty-matrix or from a JSON-file
Methods
Return Type Name Description
void InitializeContainer() Wrapper function, which sets up the container
void LoadFromJSON(string path) Loads OpenWorm NeuroML files from JSON and generates the projections of each neuron
void SetUpVertices(bool fromGraph) Sets Up Nodes and places the game objects in the scene according to the stored associated positions
void SetUpEdges(bool isStatic) Sets up LineRenderers for each connection between two nodes. The start and end points of the rendered lines are determined by the edges in the Graph. The colors of the line are determined by the type of neuron
void TogglePhysics(bool toggle) Toggles Force Directed Graph
void DestroyGraph() Clears all nodes and corresponding edges from the GraphContainer.
void ToggleNodeBodies(bool toggle) Toggles visibility of the Nodes and the corresponding edges by setting the game object active or inactive
void ChangeNodePosition() Function that is used to to interpolate and animate the position of the nodes, when a different graph view is chosen. e.g. transforming the positions from anatomical positions to the lateralized view
IEnumerator AnimateChangeNodePosition(bool force) Coroutine, which transform the position of the nodes between a start and some end point
void WrapperAnimateCoroutine(bool force) Starts AnimateChangeNodePosition(force) coroutine
void IncreaseRecursionLevel() Increments the current recursionLevel
void DecreaseRecursionLevel() Decrements the current recursionLevel
void EnableEdges(bool phys) Toggles Edges during force directed mode
void ExtractPathFromNodes(int resolution) Actually I dont recall what this was supposed to do. Sorry.
Events and Delegates
Type Name Description
delegate void OnTogglePhysics(bool phys) delegate function that is triggered, when physics are enabled
delegate void OnRecursionLevelChanged(int level) Passes the changed recursionLevel to subscribing functions
