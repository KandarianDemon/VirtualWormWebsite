Node

CyberElegans/wormbrain/Graph/Node

Description

A Node represents a vertex in a Graph and is a child object of a GraphContainer. This class is a wrapper for the NodePhysics and NodeVisuals.

The node implements the Interfaces IPointerClickHandler, IPointerExitHandler.

Properties
Type Name Description
GraphContainer graphContainer Reference to the GraphContainer the Node belongs to
NodePhysics physics Reference to the NodePhysics component of the Node
NodeVisuals visuals Reference to the NodeVisuals component of the Node
NeuronDynamics dynamics Reference to the NeuronDynamics component of the Node
GameObject[] connectedNeurons Array of GameObjecs storing all the Nodes this Node is connected to
List'<'(NeuronDynamics, float)'>' inputNodes List of Connections. float value is the strength of the connection, NeuronDynamics returns activity of neuron
Methods
Type Name Description
void Init() Initializes the Node, NodeVisuals, NodePhysics, NeuronDynamics
void SetSign() Determines the sign of the edge weight, by looking at the associated Neurontransmitters as stored in NodeProperties. -1 for inhibitory and +1 for excitatory neurotransmitters
void SetInputConnections(ref inputNodes) the inputNodes are set to the NeuronDynamics in order to calculate the neuronal activity of this Node
void SetProperties(NodeProperties properties) Links a NodeProperties object containing all the information about a neuron/node to this node
void ChangeLineSet(GameObject lineContainer) Toggles which connections are visible. Anatomical or Connectivity
void SetLineStrength(float) Changes the weight of the line drawn by the LineRenderer
Material SetLineMaterial() returns the LineMaterial of the visuals
void TogglePhysics(bool phys) Sets the physics active or inactive
void SetColliderSize(float size) Changes the size of the Collider attached to the node. Relevant for Force Directed Graph
void SubscribeAsInput(Node inputNode) Adds a node as input to this node.
void UnsubscribeFromInput(Node inputNode) Removes a node as input to this node.
float GetMembranePotential() returns current value for the membrane potential calculated by the NeuronDynamics component attached to this Node
void OnPointerClick(PointerEventData eventData) If a click event is registered on this Node, a current is injected into the NeuronDynamics component
