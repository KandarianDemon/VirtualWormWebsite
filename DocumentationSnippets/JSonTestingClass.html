CyberElegans/wormbrain/

Description

inherits from UnityEngine.Monobehaviour

This component takes NeuroML files stored in separate JSON-files and initialzes the nodes/neurons based on this data.

For each neuron a gameobject with the Node-component is initialized as the soma. Neurites are made of individual line segments. Each segment is its own gameobject with a LineRenderer-component attached to it.

Properties
Type Name Description
Material Line Material for the lines representing the segments of the neurites
List<NeuroMlCell> cells List holding the converted JSON NEuroML-files
Methods
Return Type Name Description
void LoadNeurons() Loads JSON files and converts them into NeuroMLCells. Stores all converted files in cells
void CreateConnectome() Creates GameObjects for each soma and line segments in the projection
