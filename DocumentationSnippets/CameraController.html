Camera Controller

CyberElegans/

Description

inherits from UnityEngine.Monobheviour

A camera controller that lets the user navigate the 3D scene. Takes the keyboard and mouse input and updates the transform of the camera.

Clicking on an object, e.g. Nodes/Neurons centers the camera in on that object

Properties
Type Name Description
float speed speed of the camera movement
float sensitivity sensitivity of mouse movement and rotation
Methods
Type Name Description
