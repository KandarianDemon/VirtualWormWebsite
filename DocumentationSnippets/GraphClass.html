Graph

Namespace/Path/To/Class

Description

The Graph class stores connectivity data provided in an adjacency matrix as a List of Nodes and a Dictionary of Edges.

Nodes are a string and refer to the Nodes name, where Edges are stored in a Dictionary containing a string referring to the starting node and a tuple containing the end node and the connection strength.

Properties
Type Name Description
List`<`string> Vertices Contains all the nodes in the Graph
Dictionary`<`string,List`<`(string,float)> Edges Contains all Edges. Start -> End, Strength
Int { get; } vertCount Number of Nodes
Int { get; } edgeCount Number of Edges
Methods
Return Type Name Description
void AddVertex( string vertex_name) Adds vertex to vertex list
void RemoveVertex(string vertex_name) Removes vertex from vertex list
void AddEdge( string start, string end, float strength) Adds Edge to the Edges dictionary
void RemoveEdge( string start, string end) Removes given edge from the Edges dictionary
