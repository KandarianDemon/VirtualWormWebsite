GLLines

Description

inherits from UnityEngine.Monobheviour

This class acts as a domain for the fluid simulation. The class name stems from experimenting with the GLLines API, which can be used to draw lines and shapes during runtime in the game. Here, it is used to draw the boundaries of the simulation domain.

The domain can be interacted with during runtime, meaning that its rotation, scale and position can be manipulated by the user.

Everything inside the domain is then affected by manipulations to the domain, which means you get some nice splish splashing fluids :-)

Properties
Type Name Description
Vector3 dimensions the dimensions of the domain. dimensions derive from transform.localScale
Material lineMaterial material of the lines to be drawn
readonly int[] bbox_triangles triangle indices of all tris in the bounding box as a triangle soup
Methods
Type Name Description
static void CreateLineMaterial() initializes the line material and loads the corresponding shader. Sets alpha, blend distance and culling to CullMode.Off
Vector3 GetDimensions() returns transform.localScale
void Draw() Draws the lines of the bounding box. The Bounding Box is defined in local space. Corner Points are then transformed to world space for drawing.
