Spring Physics

CyberElegans/wormbrain/MuscleSystem/Spring Physics

Description

This component implements the behaviour of a hookean spring and a hill-type model of muscle contraction and represents a muscle fibre. It is used to connect two adjacent body segmentsin order to introduce movement.

Properties
Type Name Description
GameObject spring gameobject holding the SpringPhysics component
Vector3 position1 Anchor point 1 of the spring
Vector3 position2 Anchor point 2 of the spring
Rigidbody first rigidbody of the first connected gameObject or bodySegment
Rigidbody second rigidbody of the second connected gameObject or bodySegment
BodySegment a previous Body segment
BodySegment b next Body segment
float springConstant the springs springconstant
bool isActive toggles the spring
SpringMuscle muscle muscle this spring belongs to
NeuronDynamics dynamics neuron this spring receives activation signal from, if active
CentralPatternGenerator cpg pattern generator this spring receives activation signal from, if active
float activationLevel Slider, where user can directly change the activationLevel to observe the springs response
Methods
Type Name Description
void SetPosition(Vector3 a, Vector3 b) Initializes the anchor positions of the springs and initializes the resting length
void SetRigidBodies(Rigidbody a, Rigidbody b) Sets the rigidbodies
void SetBodySegments(BodySegment a, BodySegment b, int idxA, int idxB) Sets the body segments and the correct spring anchor positions on both of the segments
void SetActivationLevel(ref float activationLevel) If theres a muscle present, its activationLevel is applied to the spring.
void SetActive(bool active) toggles isActive
void SetPassiveSpring() no activation signal is passed to the spring, therefore it does not contract, but only reacts
void IsNeckMuscle(bool neck, int med, int range) If the muscle belongs to the head and neck region of the worm, it will receive input from a gamepad instead of any other activation signals.
void SpringForce() Calculates the Force of the spring according to Hooke.
void UpdatePosition() Updates the positions of the anchors points on the spring.
void HillMuscle(float activation) implements a physiological model of muscle contraction to approximate the dynamics of a muscle fibre.
