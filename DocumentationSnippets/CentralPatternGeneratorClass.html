Central Pattern Generator

CyberElegans/wormbrain/MuscleSystem/CentralPatternGenerator

Description

This class generates a sinewave emulating a central pattern generator in the worms brain. The activity drives the muscles and is offset by muscle ID, so that a sinewave traverses the worms body.

In the current interation it also seves as an interphase to adjust spring properties or drag physics.

Properties
Type Name Description
enum SimulationMode {Hill, Spring} Sets the behaviour of the springs to hookean spring or hill-type muscle model
enum SignalOrigin {CPG, Connectome} Determines if the activation signal comes from the Central Pattern Generator or the virtual connectome
float wavelength wavelength of the sinewave
float Amplitude amplitude of the sine wave
float speed adjusts speed
float minSpringConstant minimum value for the spring constant
float maxSpringConstant maximum value for the spring constant
bool isActive toggles the central pattern generator component
float signal output signal of the central pattern generator
