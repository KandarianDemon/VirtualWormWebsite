Muscle System

CyberElegans

Description

This class is used to connect a virtual connectome to the virtual worm body. The motoneurons of the virtual signals directly drive the corresponding motoneurons.

Alternatively, the virtual connectome can be connected to a set of Model muscles which contract when activated, in order to illustrate muscular activity.
