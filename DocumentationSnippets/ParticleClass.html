Particle

Description

This is a struct which holds all information relevant to a particle. Most notably its position,velocity and density. Additional information like color, acceleration, radius, rest density of the fluid it represents can also be stored here.

The particles are then rendered as a circular billboard sprite

Properties
Type Name Description
Vector3 Color Stores assigned color as RGB-Vector
Vector3 position position of the particle
Vector3 velocity current velocity of the particle
Vector3 offset really bad naming convention: but "offset" stores the forces currently acting on the particle
float pressure represents the pressure at the particles position
float radius radius of the particle
float mass mass of the particle
int hash hash value of the particle. Indicates which cell it currently occupies
int index index of the particle in the sorted particle array
