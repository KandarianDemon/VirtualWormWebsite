Graph Utilities

Namespace/Path/To/Class

Description

The Graph Utilities class is a static class, that contains some methods to analyze Graph Objects. Among these functions are an implementation of the Dijkstra-algorithm to find the shortest path between two vertices.

It also contains methods to turn a given adjacency matrix into a Graph object, and subsequently into a GraphContainer

Methods
Return Type Name Description
static void Dijkstra(Graph graph, string start, string end) Applies Dijkstras' algorithm to find the shortest path between two nodes
static void AdjacencyMatrixToGraph(string filepath, Graph graph) Uses CSVReader() to read connectivity data stored in a csv-file and stores this information in a Graph object
static Dictionary`<`string,NodeProperties> PropertiesFromGraph(Graph graph) Generates a dictionary of Nodes and their corresponding properties. The property-objects are empty and must be filled separately
static void SetNodePosition(Dictionary`<`string, NodeProperties> props, Graph graph, string positionFilePath, float scale, Transform transform) Gets 3D position as specified in a corresponding csv-file and initializes 3D transforms representing the respective node at the corresponding position
static void SetNodeLocation(Dictionary`<`string, NodeProperties> props, Graph graph, string positionFilePath, float scale, Transform transform) Looks at Node Properties and gets the name of the Node. The position of the node is offset in y- or x-direction in regard to the neuron being lateralized left,right, dorsally or ventrally
static void SetNodeColor(Dictionary`<`string, NodeProperties> props, Graph graph, string positionFilePath, float scale, Transform transform) Determines the color of the Node representing a neuron. Options are: Sensory neurons, interneurons, motorneurons, misc
