NodeVisuals

CyberElegans/ wormbrain/ Graph/ NodeVisuals

Description

The Node Visual-class handles anything related to the visual appearance of a Node. E.g. if any interaction events occur, the Nodes color and size are changed to indicate that it is currently in focus. Depending on the GraphContainers recursionLevel, downstream connected Nodes/neurons are also triggered to be highlighted.

Properties
Type Name Description
MeshRenderer renderer Unity-class. Renders mesh data as a 3D object
Material material Material and shader, which is applied to the mesh stored in renderer
GameObject nodeBody GameObject which holds the mesh representing the node. In this case the nodes are rendered as spheres, but it can be any arbitrary shape.
List<`Node> connectedNodes List containing all the nodes connected to this Node
List`<`LineRenderer> lines List of all the gameobjects parented to this node, which contain a connection between two nodes. LineRenderers draw the corresponding lines
Material LineMaterial Material applied to the respective lines. Uses a custom lineShader
float lw represents the lineweigth
float radius radius of the sphere representing the Node body
Methods
Type Name Description
void ChangeSize(float size) Changes size of the object by the supplied value
void ChangeColor(Color newColor) Changes color of the node to the provided color
void ChangeLineSet(GameObject container) Changes the lines representing the connections. Either anatomically connections, or connectivty can be chosen
void Reset() Resets all changed values to the default ones
void SetColor(Color color) Sets the default color
void UpdateLines(Rigidbody start, Rigidbody end) Updates the position of the Lines being drawn. Especially important if physics is true during Force Directed Mode
void InitConnectedNodes() Subscribes the Recursion() method of the connected nodes to the pointerEnter-event.
void InitLines(GameObject container) Adds all lines in the container to the lines List
void ClearLines() clears all entries from the lines-list
void Recursion(int level, ref onPointerExit exit) Recursive function that decreases the provided recursion level. Adds connected nodes to the pointerExit event. Every connected node within recursion level is highlighted along with the respective connections
void SetRecursionLevel(int level) Sets the recursionLevel of this node to the provided level
